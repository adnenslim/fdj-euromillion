## Installation and Setup Instructions

```
git clone https://github.com/adnenslim/fdj-euromillion.git
cd fdj-euromillion
yarn
```

## Running the app

To get the app up and running (and really see if it worked), run:

```
yarn start
```

## To Lint

```
yarn lint
```

### To Fix Lint

```
yarn lint-fix
```

## To Build

```
yarn build
```
